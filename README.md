# Assignment

-> **To run this file first you have to install exceljs library which is a javascript library to do CRUD operation in Spreadsheet.**

-> Command to install exceljs library is :  
        ~ ***npm install exceljs --save***
        
-> Then you have to create a new npm package file by writing this command : 
                ~ ***npm init -y***

-> Then you have to place the spreadsheet in the same root folder in which you are working

-> Now to run the file, the following command is :  
           ~ ***node main.js***
